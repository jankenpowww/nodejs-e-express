#01 - O que é NodeJS?

O NodeJS, ainda Node.js ou somente Node é um interpretador de JavaScript que permite programas escritos na linguagem rodarem em ambientes fora do navegador. Como sabemos, JavaScript é uma linguagem interpretada, o que significa que ela não depende de um compilador para rodar um código assim como a grande maioria das outras linguagens como Python ou Java, por exemplo. Um interpretador de JavaScript que roda fora dos navegadores dá a linguagem a capacidade de ser utilizada para outros fins além das interações e da sua função de ser uma lingua propriamente feita para o front-end, e é isso o que o NodeJS faz.

O NodeJS ainda possui seu próprio gerenciador de pacotes, assim como o Python possui o pip e o C# possui o NuGet: o npm. Através do npm, você consegue baixar e instalar soluções prontas e implementá-las no seu aplicativo de forma significativamente fácil, através de poucos digitos no terminal. Em resumo, com o NodeJS e o npm será possível adicionar mais artifícios e recursos para seu projeto e permitir você desenvolver a parte backend de uma aplicações utilizando JavaScript! O NodeJS

O NodeJS nada mais é do que um ambiente de terminal que roda JavaScript. Sendo assim, você pode escrever "pequenos trechos de JavaScript" no terminal e até testar alguns métodos conhecidos, como o famigerado console.log(). Quem sabe, declarar variáveis no processo? Tudo isso é possível no terminal do Node e comprovar que sim, estamos rodando JavaScript fora do navegador!


> console.log("Olá, mundo!")
Olá, mundo!

> let nome = "Júlio"
undefined

> console.log("Meu nome é " + nome + ".")
Meu nome é Júlio.

Perceba que o ambiente de terminal acima descreve todas as entradas de comando e saída dentro do terminal do node. Sempre que quisermos iniciar um novo terminal do node, escrevemos 'node' e apertamos enter em seguida.

É importante ressaltar isso quando quisermos verificar algumas coisas a respeito do próprio node instalado na máquina, então o comando 'node' não vai existir dentro do próprio terminal do Node.

No prompt de comando/cmd, se quisermos verificar a versão atual instalada do Node, utilizamos o seguinte comando: 'node -v'.

Mas, se quisermos fazer essa verificação dentro do terminal do Node, digitamos o seguinte:


> process.version
'v17.0.0'

process nada mais é que um objeto que contém todas as informações à respeito da versão atual do NodeJS instalada na máquina.
