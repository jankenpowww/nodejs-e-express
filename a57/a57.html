a57 - Formulário de Login

Criando o formulário de login

Agora que já podemos ter usuários cadastrados na nossa aplicação, vamos partir para o próximo passo: desenvolver o recurso de login e autenticação de usuário.

Nesta aula, vamos desenvolver primeiramente a página de login, onde tudo isso será possível.



Definindo rota e view

Vamos primeiramente definir a rota de onde o formulário pode ser acessado, dentro do arquivo de grupo de rotas do usuário:

usuario.js

...

router.get("/login", (req, res) => {
    res.render("usuario/login")
})

module.exports = router

E então criar o arquivo 'login.handlebars', dentro do diretório 'views/usuario'.

login.handlebars

<h2>Login</h2>
<small>Entre agora mesmo e acesse os conteúdos.</small>
<hr>
<div class="card">
    <div class="card-body">
        
        <form method="post" action="/usuario/login">
            <label>E-mail:</label>
            <input type="email" class="form-control mb-3" name="email" required>

            <label>Senha:</label>
            <input type="password" class="form-control mb-4" name="senha" required>

            <button style="width: 100%;" class="btn btn-success" type="submit">Logar</button>
        </form>
        
    </div>
</div>

Na próxima aula, vamos ver como montar a autenticação do usuário com o Passport, um módulo do npm que permite montar estratégias de autenticação para aplicações feitas com NodeJS.
