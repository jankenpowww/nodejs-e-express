Template HTML para conteúdo de aulas

Como utilizar?

Aqui vão os textos de cada seção. Os textos em HTML possuem algumas tags de formatação padrões e que podem ser utilizados, como destacar um texto, transformá-lo em itálico, sublinhá-lo, rasurá-lo ou utilizar a tag de marcação específica para marcá-lo. Isso sem contar as tags de âncora para linkagem de páginas em texto comum.

Envolvendo a tag <span> em alguma parte do texto e uma classe específica descrita nas regras de estilo deste documento pode prover outros estilos prontos. Observe:

  • .tVermelho - torna o texto vermelho.
  • .tLaranja - torna o texto laranja.
  • .tVerde - torna o texto verde.
  • .tAzul - torna o texto azul.
  • .tRoxo - torna o texto roxo.
  • .tRosa - torna o texto rosa.
  • .tCinza - torna o texto cinza.
  • .tBranco - torna o texto branco, lol.

Para exibir comandos de código no meio de um texto, você pode utilizar a classe "tCodigo" envolvido numa tag <span>. Exemplo: o comando isNaN() retorna 'true' apenas se a informação de uma variável não for numérico.

Veja na próxima seção sobre as estruturas existentes que podem ser utilizadas...



Estruturas do documento

Você pode utilizar tabelas para catalogar elementos e suas descrições.

Comando Descrição
alert() Exibe uma janela de alerta no topo da página.
prompt() Exibe uma janela com um campo de texto.
confirm() Abre uma janela pedindo confirmação do usuário.

É possível também incorporar trechos de código...


<!-- Estrutura básica de um documento HTML. -->
<!DOCTYPE html>
<html lang="pt-BR">
    <head>
        <title>Título do Documento</title>
    </head>
    <body>
        <p>Isso é um parágrafo.</p>
    </body>
</html>

As estruturas de documento podem ser utilizadas quando há a necessidade de exemplificar coisas que não necessariamente funcionam em formato de texto comum, vide amostras de código-fonte, por exemplo.
